# 🚀 Inicio Rápido

## Instalación (1 minuto)

```bash
npm install
npm start
```

¡Listo! La aplicación se abrirá en tu navegador.

## Primera Vez Usando la App

### 1. Crear un Hábito
- Click en **"+ Nuevo Hábito"**
- Completa los campos
- Click en **"Crear Hábito"**

### 2. Marcar como Completado
- Click en **"Completar Hoy"** en la tarjeta
- Se guardará automáticamente

### 3. Ver Progreso
- Click en **"Ver Calendario"**
- Los días verdes = completado

### 4. Si Cometiste un Error
- Click en **"↶ Deshacer"**

## Preguntas Frecuentes

**¿Dónde se guardan mis datos?**
- En el almacenamiento local de tu navegador (localStorage)
- Los datos se guardan automáticamente
- Persisten incluso si cierras el navegador

**¿Puedo exportar mis datos?**
- Sí, click en **"📥 Exportar"**
- Se descargará un archivo JSON

**¿Qué pasa si limpio mi caché?**
- Se perderán los datos
- Puedes importar desde tu archivo JSON

**¿Cuánto almacenamiento tengo?**
- Aproximadamente 5-10 MB
- Suficiente para miles de hábitos

## Accesos Directos

| Acción | Botón |
|--------|-------|
| Crear hábito | + Nuevo Hábito |
| Deshacer última acción | ↶ Deshacer |
| Exportar datos | 📥 Exportar |
| Eliminar todo | 🗑️ Limpiar Todo |
| Marcar hoy | Completar Hoy |
| Ver calendario | Ver Calendario |

## Tips y Trucos

✨ **Racha**: El número 🔥 muestra tu racha actual de días consecutivos

📊 **Tasa**: El porcentaje indica tu tasa de cumplimiento desde que creaste el hábito

📅 **Calendario**: Puedes hacer click en cualquier día para marcar/desmarcar

🎨 **Colores**: Cada hábito puede tener su propio color para identificarlo rápidamente

## Limpieza de Datos

Si necesitas empezar de cero:

1. Click en **"🗑️ Limpiar Todo"**
2. Confirmar en el diálogo
3. Se eliminarán todos los hábitos

**Nota**: Esto NO se puede deshacer, así que ten cuidado 😊
